## Frontend Setup

#### Change directory

```
cd fe
```

#### Install packages
```
 npm install
```

#### Start frontend server
```
npm run serve
```
 

## Backend Setup

#### Change directory

```
cd be
```

#### Setup Enviorement
```
python -m venv test-env
```

#### Active Enviorement
```
 test-env\Scripts\active
```

#### Install packages
```
pip install -r requirements.txt
```
#### Start Server
```
python main.py
```
